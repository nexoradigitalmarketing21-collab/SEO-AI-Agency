# Client Onboarding Checklist

## Phase 1: Initial Contact

- [ ] Receive lead inquiry via website form
- [ ] Send welcome email with service overview
- [ ] Schedule strategy call
- [ ] Send calendar invite

## Phase 2: Discovery & Assessment

- [ ] Conduct 30-minute strategy call
- [ ] Understand client goals and KPIs
- [ ] Review current website/SEO status
- [ ] Document competitor analysis
- [ ] Identify technical issues
- [ ] Estimate project timeline

## Phase 3: Proposal & Agreement

- [ ] Generate custom SEO proposal
- [ ] Create service agreement
- [ ] Send via client portal
- [ ] Collect digital signature

## Phase 4: Payment Setup

- [ ] Create Stripe checkout session
- [ ] Collect payment method
- [ ] Confirm subscription plan
- [ ] Send payment receipt

## Phase 5: Project Kickoff

- [ ] Create project in system
- [ ] Assign team members
- [ ] Share project timeline
- [ ] Set up reporting schedule
- [ ] Grant portal access

## Phase 6: First Week Actions

- [ ] Complete technical audit
- [ ] Install tracking codes
- [ ] Submit initial keyword list
- [ ] Create content calendar
- [ ] Schedule monthly check-in

---

## Ongoing Monthly Process

1. Keyword position tracking
2. Content optimization recommendations
3. Link building outreach
4. Technical maintenance
5. Monthly report generation
6. Client call & strategy review
