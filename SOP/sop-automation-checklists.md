# SOP Automation Checklists

## Purpose

This document provides standardized checklists for every major SEO workflow. Each agent follows the same proven process to ensure consistency, quality, and efficiency.

**Benefits:**
- Consistent execution across all projects
- Quality assurance at every step
- Easy training for new agents
- Scalable processes
- Reduced errors and omissions

---

## How to Use These Checklists

### For Agents
1. Open the relevant checklist for your task
2. Check off each item as you complete it
3. Don't skip steps - follow in order
4. Submit to QA only when all items are checked
5. Update checklist with any issues found

### For QA Agent
1. Verify checklist completion
2. Check that all items were completed
3. Review quality of each item
4. Approve or request revisions
5. Document any process improvements
