# Keyword Research Agent

## Purpose

The Keyword Research Agent specializes in **comprehensive keyword analysis, search intent identification, and content opportunity discovery**. It uncovers valuable keyword opportunities, analyzes search intent, and provides data-driven insights to guide content and SEO strategy.

**The Keyword Research Agent is the "detective" of the agency** - uncovering what users are searching for and why.

## Keyword Research Agent Capabilities

### Keyword Discovery

**What Keyword Research Agent Does:**
- Uses multiple tools and methods
- Expands seed keywords
- Discovers long-tail opportunities
- Identifies question keywords
- Finds related terms and synonyms

**Discovery Methods:**
- Tool-based research (Ahrefs, SEMrush, Moz)
- Google autocomplete
- Google Search Console data
- Competitor analysis
- Customer feedback
- Industry forums and Q&A sites
- AnswerThePublic
- Google Trends

### Search Intent Analysis

**What Keyword Research Agent Analyzes:**
- User intent behind queries
- Content format preferences
- Buyer journey stage
- User needs and expectations
- Conversion potential

**Intent Types:**
- **Informational:** "how to", "what is", "guide"
- **Navigational:** Brand names, specific sites
- **Transactional:** "buy", "price", "discount"
- **Commercial:** "best", "review", "vs", "comparison"

### Keyword Analysis

**What Keyword Research Agent Evaluates:**
- Search volume (monthly searches)
- Keyword difficulty (ranking challenge)
- Cost per click (commercial value)
- Current rankings
- Competition level
- Trend direction
- Seasonal patterns

### Content Opportunity Identification

**What Keyword Research Agent Identifies:**
- Content gaps (keywords competitors rank for, but client doesn't)
- Topic clusters (related keyword groups)
- Quick win opportunities (easy to rank keywords)
- High-value targets (high volume, low difficulty)
- Strategic keywords (aligned with business goals)

**Phase 5: Delivery (Days 9-10)**
- QA review
- Final revisions
- Client delivery
- Presentation and walkthrough
- Implementation support
