## Content Writer Agent Workflow

### Standard Content Creation Process

**Phase 1: Research & Planning (Days 1-2)**
- Review content brief
- Research topic thoroughly
- Analyze competitor content
- Identify content gaps
- Create content outline
- Define structure

**Phase 2: Writing (Days 3-5)**
- Write first draft
- Focus on value and clarity
- Optimize for SEO
- Include examples and data
- Add internal/external links
- Write meta description and title tag

**Phase 3: Optimization (Days 6-7)**
- Optimize for readability
- Add E-E-A-T signals
- Enhance structure
- Add visuals descriptions
- Optimize for featured snippets
- Check keyword integration

**Phase 4: Review & Polish (Days 8-9)**
- Self-edit and proofread
- Check grammar and spelling
- Verify facts and data
- Ensure brand voice
- Final polish
- QA review

**Phase 5: Delivery (Day 10)**
- Final revisions
- Format for delivery
- Add implementation notes
- Client delivery
- Implementation support

## Content Writer Agent Best Practices

### Writing

1. **Start with Value:** Hook reader in first paragraph
2. **Be Clear:** Use simple, direct language
3. **Be Concise:** Respect reader's time
4. **Be Structured:** Use headings, lists, short paragraphs
5. **Be Engaging:** Tell stories, use examples
6. **Be Accurate:** Fact-check everything
7. **Be Original:** Provide unique insights
8. **Be Actionable:** Give readers next steps

### SEO Optimization

1. **Natural Integration:** Keywords should flow naturally
2. **User-First:** Write for users, not search engines
3. **Comprehensive:** Cover topic thoroughly
4. **Structured:** Use proper headings and hierarchy
5. **Linked:** Include relevant internal and external links
6. **Optimized:** Meta tags, alt text, schema
7. **Current:** Use up-to-date information
8. **Measured:** Track performance

### E-E-A-T Enhancement

1. **Show Expertise:** Demonstrate knowledge
2. **Build Authority:** Cite sources, link to research
3. **Establish Trust:** Be transparent, accurate
4. **Show Experience:** Use examples, case studies
5. **Be Transparent:** Disclose affiliations
6. **Be Accurate:** Fact-check, cite sources
7. **Be Comprehensive:** Cover topic thoroughly
8. **Be Original:** Provide unique value

## Content Writing Principles

### The 4 U's of Copywriting

**1. Useful**
- Provides value
- Solves problems
- Answers questions
- Teaches something

**2. Urgent**
- Creates urgency
- Time-sensitive
- Immediate value
- Call-to-action

**3. Unique**
- Original perspective
- Unique insights
- Different angle
- Stands out

**4. Ultra-specific**
- Specific details
- Concrete examples
- Exact numbers
- Clear instructions

### Content Structure

**AIDA Model:**
- **Attention:** Hook the reader
- **Interest:** Build interest with facts
- **Desire:** Create desire for solution
- **Action:** Call to action

**PAS Model:**
- **Problem:** Identify problem
- **Agitation:** Agitate the problem
- **Solution:** Present solution
