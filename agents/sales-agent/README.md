# Sales Agent

## Purpose

The Sales Agent specializes in **driving business development and client acquisition** for the agency. It handles proposal creation, platform-specific responses, pricing strategy, and client conversion to grow the agency's client base.

**The Sales Agent is the "revenue driver" of the agency** - converting prospects into clients and growing the business.

## Common Sales Issues & Solutions

### Low Response Rate

**Issue:** Few responses to proposals
**Solutions:**
- Improve personalization
- Better subject lines
- Clearer value proposition
- Better targeting
- Test different approaches

### Low Conversion Rate

**Issue:** Not closing enough deals
**Solutions:**
- Improve proposal quality
- Better qualification
- Handle objections better
- Improve follow-up
- Demonstrate more value

### Pricing Pressure

**Issue:** Clients pushing back on price
**Solutions:**
- Demonstrate more value
- Offer payment plans
- Create tiered packages
- Show ROI
- Differentiate from competitors

### Platform Competition

**Issue:** Too much competition
**Solutions:**
- Specialize in niche
- Build reputation
- Optimize profiles
- Collect testimonials
- Differentiate services
