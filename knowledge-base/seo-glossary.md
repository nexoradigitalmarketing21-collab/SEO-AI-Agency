# SEO Glossary

A comprehensive glossary of SEO terms and definitions used throughout the Nexora AI SEO Agency.

## A

### Algorithm
A complex set of rules and calculations used by search engines to determine the relevance and ranking of web pages.

### Alt Text (Alternative Text)
Descriptive text added to images on web pages that helps search engines understand the image content and improves accessibility.

### Anchor Text
The clickable text in a hyperlink. SEO-friendly anchor text is relevant to the page being linked to.

## B

### Backlink
A link from one website to another. Backlinks are crucial for SEO as they signal authority and trust to search engines.

### Black Hat SEO
Unethical SEO techniques that violate search engine guidelines and can result in penalties.

## C

### Canonical Tag
An HTML element that tells search engines which version of a page is the "master" version when there are duplicate pages.

### Click-Through Rate (CTR)
The percentage of users who click on a search result after seeing it.

### Content Marketing
Creating and sharing valuable content to attract and engage a target audience.

## E

### E-E-A-T
Experience, Expertise, Authoritativeness, and Trustworthiness - Google's quality rating framework.

## I

### Indexing
The process by which search engines store and organize web pages for quick retrieval during searches.

## K

### Keyword
A word or phrase that users enter into search engines to find relevant content.

### Keyword Density
The percentage of times a keyword appears on a page compared to the total word count.

## L

### Long-Tail Keyword
A specific, longer keyword phrase (usually 3+ words) that targets niche searches with lower competition.

## M

### Meta Description
A brief summary of a web page's content that appears in search results below the title.

### Meta Tags
HTML tags that provide metadata about a web page to search engines.

## N

### Noindex
A meta tag or HTTP header that tells search engines not to index a page.

### Nofollow
An attribute added to links that tells search engines not to pass link equity.

## O

### Off-Page SEO
SEO activities performed outside of your website, such as link building and social media marketing.

### On-Page SEO
Optimization techniques applied directly to web pages, including content, HTML, and site architecture.

## P

### PageRank
Google's algorithm that measures the importance of web pages based on the quantity and quality of backlinks.

### Pay-Per-Click (PPC)
An advertising model where advertisers pay each time a user clicks on their ad.

## R

### Rank
The position a webpage appears in search engine results for a specific query.

### Robots.txt
A file that tells search engine crawlers which pages or files they can or cannot request from your site.

## S

### Schema Markup
Structured data that helps search engines understand the content of a page and can enhance search results with rich snippets.

### Search Engine
A web-based tool that helps users find information on the internet (e.g., Google, Bing).

### Search Intent
The reason behind a user's search query - informational, navigational, transactional, or commercial.

### SERP (Search Engine Results Page)
The page displayed by a search engine in response to a user's query.

### Sitemap
A file that lists all the pages on a website to help search engines crawl and index the site more effectively.

## T

### Technical SEO
The process of optimizing a website's technical aspects to improve search engine visibility.

### Title Tag
An HTML element that specifies the title of a web page, displayed as the clickable headline in search results.

## U

### URL (Uniform Resource Locator)
The address of a web page or resource on the internet.

## W

### Web Crawler
A bot that systematically browses the web to index pages for search engines.

### White Hat SEO
Ethical SEO techniques that follow search engine guidelines and focus on providing value to users.
